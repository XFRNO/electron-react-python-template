# Frontend Tech Stack

This document provides an overview of the technologies used in the frontend application.

## Core Technologies

- **React**: A JavaScript library for building user interfaces.
- **Vite**: A fast build tool and development server for modern web projects.
- **TypeScript**: A typed superset of JavaScript that compiles to plain JavaScript.

## UI and Styling

- **Tailwind CSS**: A utility-first CSS framework for rapidly building custom designs.
- **Shadcn UI**: A collection of re-usable components built using Radix UI and Tailwind CSS.
- **TanStack Router**: A fully type-safe router for React.

## State Management

For state management, we recommend using a combination of React's built-in hooks (`useState`, `useReducer`, `useContext`) for local state and a library like Zustand or Redux Toolkit for global state if needed.

## Tooling

- **pnpm**: For package management.
- **ESLint**: For code linting.
- **Prettier**: For code formatting.
