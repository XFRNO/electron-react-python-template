# Backend Tech Stack

This document provides an overview of the technologies used in the backend application.

## Core Technologies

- **Python**: A versatile and powerful programming language.
- **FastAPI**: A modern, fast (high-performance), web framework for building APIs with Python 3.7+ based on standard Python type hints.
- **Uvicorn**: A lightning-fast ASGI server, used to run the FastAPI application.

## Package Management

- **uv**: A fast Python package installer and resolver, written in Rust. It's used for managing the Python dependencies.
- **pnpm**: Used to run scripts and manage the workspace.

## Tooling

- **Pytest**: A mature full-featured Python testing tool.
- **Black**: The uncompromising Python code formatter.
