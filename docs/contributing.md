# Contributing

We welcome contributions from the community! Whether you're fixing a bug, adding a new feature, or improving the documentation, your help is greatly appreciated.

## Getting Started

1.  **Fork the repository**: Click the "Fork" button at the top right of this page.
2.  **Clone your fork**: `git clone https://github.com/YOUR_USERNAME/electron-react-python-template.git`
3.  **Create a branch**: `git checkout -b my-awesome-feature`
4.  **Make your changes**: Make your changes and commit them with a clear and concise commit message.
5.  **Push your changes**: `git push origin my-awesome-feature`
6.  **Open a pull request**: Go to the original repository and open a pull request.

## Development

To get started with development, you'll need to have `pnpm` installed. Then, you can install the dependencies and run the development server:

```bash
pnpm install
pnpm dev
```

This will start the Electron app, the React frontend, and the Python backend in development mode.

## Code Style

Please ensure that your code adheres to the existing code style. We use ESLint for JavaScript/TypeScript and Black for Python.

## Submitting a Pull Request

When you're ready to submit a pull request, please make sure that:

- Your code is well-tested.
- Your commit messages are clear and descriptive.
- You've updated the documentation if necessary.

Thank you for your contributions!
